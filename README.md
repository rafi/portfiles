Rafi's Portfiles
===
My local Macports repository.

Install
---
1. Clone somewhere
2. Add the path to your `/opt/local/etc/macports/sources.conf` e.g. `file:///Users/rafi/code/portfiles`
3. Change directory to the cloned repository, and run `portindex`
4. Enjoy!
